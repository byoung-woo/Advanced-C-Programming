# 🚀 Advanced C Programming Lab

C언어의 심화 문법을 학습하고, 이를 활용해 수치 해석, 문자열 조작, 파일 시스템 제어 및 이미지 처리 알고리즘을 구현한 과제 및 시험 문제 풀이 저장소입니다.

## 📂 디렉토리 구성 및 주요 학습 내용

### 1) 중간고사 실습 (`01_middle/`)
C언어의 기초 연산과 제어문, 다차원 배열 및 함수 활용 능력을 다룹니다.

- **수치 적분 프로그램 (`no1.c`)**  
  사각형 방법을 이용하여 선형 함수 `f(x)=ax`, 2차 함수 `f(x)=ax^2+1`, 사인 함수 `f(x)=a·sin(x)`의 정적분 값을 계산하는 로직을 구현했습니다.

- **랜덤 샘플링 및 통계 (`no2.c`)**  
  주어진 데이터 배열에서 무작위로 값을 추출하여 평균을 산출하고, 이를 통해 훈련 데이터(`trData`)를 생성하는 시뮬레이션을 수행했습니다.

- **행렬 곱셈 구현 (`no3.c`)**  
  `3×2` 행렬과 `2×3` 행렬의 곱셈 알고리즘을 함수화하여 구현하고, 다차원 배열의 메모리 참조 방식을 학습했습니다.

---

### 2) 기말고사 실습 (`02_final/`)
구조체, 포인터, 파일 입출력 및 이진 데이터 처리를 포함한 심화 응용 능력을 다룹니다.

- **문자열 검색 및 변환 (`No1.c`)**  
  문장에서 특정 단어를 찾아 대문자로 변환(Capitalize)하는 로직을 구현했습니다.  
  `strstr`, `strncat` 등을 활용한 문자열 조작을 익혔습니다.

- **구조체 성적 관리 시스템 (`No2.c`)**  
  `struct`로 학생 정보를 관리하고, 성적순 정렬(Sorting) 및 조건에 따른 Pass/Non-Pass(P/NP) 판정 로직을 구현했습니다.

- **파일 텍스트 분석기 (`No3.c`)**  
  텍스트 파일을 읽어 라인 수, 단어 수, 공백 제외 문자 수를 카운트하고, 대소문자 구분 없이 특정 단어 빈도수를 측정하는 유틸리티입니다.

- **PPM 이미지 병합 도구 (`No4.c`)**  
  이진 파일 입출력으로 두 개의 PPM(Portable Pixmap) 이미지를 읽어 가로로 하나로 합치는 기초 이미지 프로세싱을 실습했습니다.
